# VoiceNote v1.0.0 リリースチェックリスト

## ビルド前の確認

- [ ] デバッグ用のprint/NSLog文がすべて除去されていること

- [ ] Bundle Identifier が正しいこと（例: tokyo.underbar.VoiceNote）
- [ ] バージョン番号が 1.0.0 になっていること
- [ ] Deployment Target が macOS 14.0 になっていること

## 機能テスト

- [ ] 右⌘×2 ホットキーで録音開始/停止
- [ ] Apple音声認識（高速モード）で書き起こし → 自動ペースト
- [ ] Whisper音声認識（高精度モード）で書き起こし → 自動ペースト
- [ ] 設定画面でエンジン切替 → 次回録音時に反映
- [ ] 音声コマンド（見出し、項目、改行）が動作すること
- [ ] クリップボードコピーのみ（自動ペーストOFF時）
- [ ] メニューバーアイコンの状態変化（待機/録音中/処理中）
- [ ] 設定画面の各タブが正常に表示されること
