VoiceNote とは

VoiceNote は macOS のメニューバーに常駐する音声入力アプリです。右⌘キーを2回タップするだけで録音が始まり、話した内容をテキストに変換してアクティブなアプリに自動ペーストします。

2つの認識エンジンを切替可能

VoiceNote には2つの音声認識エンジンを搭載しており、用途に応じて切り替えられます。

Whisper（高精度モード）

OpenAI の Whisper Large V3 Turbo を WhisperKit 経由で Apple Silicon 上にオンデバイス実行。専門用語や英単語が混じった内容も高い精度で認識します。ただしモデルが大きいため、処理にはやや時間がかかります。初回使用時に約3GBのモデルをダウンロードする必要があります。精度を重視したい場面に向いています。

Apple 音声認識（高速モード）

macOS 内蔵のオンデバイス音声認識を使用。モデルのダウンロードが不要で、Whisper に比べて処理が高速です。短いメモやチャット入力など、速度を重視したい場面ではこちらが便利です。

設定画面からワンタッチで切り替えられるので、場面に応じて使い分けてください。

v2.1 の新機能

🌐

Bridge Send — 日→英翻訳送信

日本語で話した内容を、その場で英語に翻訳して送信。原文と訳文を2ペインのオーバーレイで確認・編集でき、プリセット切替（⌃⌥P）で「EN-Mail」「EN-Chat」など用途に応じた翻訳スタイルを選択可能。翻訳には Apple Translation framework を使用（macOS 15+、オフライン対応）。

⚡

Quick Translate — 選択テキスト翻訳

テキストを選択して ⌃⌥T を押すだけ。ブラウザでもメールでもエディタでも、選択テキストを日↔英で即時翻訳表示します。Bridge Send が「話して翻訳」なら、Quick Translate は「選んで翻訳」です。

📖

用語集（Glossary）

プロジェクトごとに専門用語と訳語のペアを登録しておくと、翻訳時に自動反映。社内用語や技術用語の訳語を統一できます。

⌨️

ショートカットカスタマイズ

録音開始・停止、Bridge Send、Quick Translate、プリセット切替など主要な操作のキーバインドを自由に変更可能。設定は再起動後も保持されます。

音声コマンドで Markdown 入力

VoiceNote では、声だけで Markdown 記法を入力できます。

「見出し」と言えば ## が、「大見出し」なら # が入力されます。「項目」で箇条書きの - が、「改行」で改行が入ります。「太字」「ボールド」で ** が挿入されるなど、多数の音声コマンドに対応しています。

議事録やメモを声で取るとき、構造化された Markdown ドキュメントをハンズフリーで作成できるのが大きな特徴です。

アプリルプロファイル

VoiceNote はアクティブなアプリを検知して　出力フォーマットを自動調整します。

Slack や Discord にペーストする場合は Markdown 記法を除去したプレーンテキストで出力。VSCode や Xcode ではコード入力に適したインデント調整を行います。アプリごとに末尾改行の追加有無なども細かく制御されています。

「どのアプリで使っても違和感なく入力できる」を目指した機能です。

Gemini API 連携（BYO API）

🔧 オプション機能 — 設定画面から Gemini API キーを登録すると、以下の機能が強化されます。未設定でも音声入力・翻訳の基本機能はすべてオンデバイスで動作します。

テキスト整形

音声書き起こ後のテキストを、Gemini（gemini-2.0-flash）が自動整形します。フィラー（えー、あのー等）の除去、句読点の補正、長文の場合は見出しや管条書きでの構造化まで行います。話し言葉のニュアンスは残しつつ、AI チャットへの入力に適した形に整えてくれます。

Bridge Send の文体調整

プリセット（EN-Mail、EN-Chat 等）を選んだ際に、翻訳後の英文をプリセットに応じたトーンに自動調整します。

トークン保護

Bridge Send や Quick Translate で翻訳する際、URL、メールアドレス、コードブロック、ファイルパス、コマンド（$ で始まる行）、バージョン番号や Git ハッシュなどを自動的にプレースホルダで保護し、翻訳後に復元します。

技術的な内容を翻訳するとき、URL が崩れたりコードが意味不明に翻訳されたりする問題を防ぎます。保護対象は設定画面から個別にオン・オフできます。

こんな場面で使えます

💬 AI チャットへのプロンプト入力

ChatGPT や Claude にプロンプトを打ち込む代わりに、考えていることをそのまま話す。長文プロンプトもストレスフリーに入力できます。

📧 英語メールの作成

日本語で内容を話し、Bridge Send で英語に翻訳。プリセットを「EN-Mail」に切り替えれば、ビジネスメール向けのトーンで翻訳されます。

📝 議事録・メモの高速入力

会議中にリアルタイムでメモを取る。音声コマンドで見出しや管条書きの Markdown 記法も入力可能です。

💻 コーディング中のドキュメント入力

Xcode や VSCode でコードを書きながら、コメントやドキュメントを声で入力。アプリ別プロファイルでフォーマットが自動調整されます。

🔒 プライバシー重視 — 音声データはすべてデバイス《で処理されます。Whisper も Apple Translation framework も完全オンデバイス。音声が外部サーバーに送られることはありません。唯一の例外は Gemini API キーを設定した場合のテキスト整形・文体調整で、この場合のみ書き起こし後のテキスト（音声データではなく文字データ）が Gemini API に送信されます。
